GREEN MATH

It is simple interactive game developed in HTML5 using jQuery UI. It is simple project to learn jQuery, timer, delay and animation as well as jQuery UI draggable, droppable, effects and button.

About Game:

Using basic airthmetic fundamentals addition, substraction, multiplication and division, try to solve problems in 30 seconds. Result number which will be randomly generated and it should match with your answer. If your answer is wrong then spider will be down towards green animal. The basic funda in this game is that try to solve max problems to avoid spider to move towards the animal and make good score points.
